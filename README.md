# 🌿 Finca Santa Fe — Sitio Web Oficial

Sitio web oficial de **Finca Santa Fe**, productora artesanal de quesos ubicada en Coronado, San José, Costa Rica.

---

## 📋 Descripción del proyecto

Este proyecto fue desarrollado como parte del **Proyecto Integrador** y los cursos de **Diseño Web**, **Programación 1** y **Programación 2** de la universidad.

El objetivo es construir una aplicación web full stack que permita a los clientes conocer los productos de la finca y realizar pedidos en línea a través de WhatsApp.

---

## 🧀 Productos

- Mozzarella
- Palmito
- Pizzero
- Semiduro
- Tierno
- Cremoso

## 🛠️ Stack tecnológico

| Capa | Tecnología |

| Backend (Fase 2) | Node.js + Express |

| Control de versiones | Git + GitHub |
